Merge Sort Algorithm - Analysis

Lets set up the recurrence relationship.

  C(n) = 2C(n/2) + Cmerge(n) for n > 1, C(1) = 0.
  Cmerge(n) = n − 1
  C(n) = 2C(n/2) + n − 1 for n > 1, C(1) = 0.

How do we solve this recursion? There are several ways. You can use the forward/backward substitution method, use a recursion tree, or use the Master Theorem (which we will see shortly).
