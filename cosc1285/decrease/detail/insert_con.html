Insertion Sort

Insertion sort is the method people often use to sort a hand of playing cards. The basic idea:

  • Consider each element one at a time.
  • Insert each element in its proper place among those already considered. (i.e. Insert into a already sorted sub-file).
  • The current element being considered is inserted by moving larger elements one position to the right and then inserting the element into the vacated position.
  ALGORITHM InsertionSort (A[0 . . . n − 1])
/* Sort an array using an insertion sort. */
/* INPUT : An array A[0 . . . n − 1] of orderable elements. */
/* OUTPUT : An array A[0 . . . n − 1] sorted in ascending order. */
    for i = 1 to n−1 do
      v= A[i]
      j= i−1
      while j ≥ 0 and A[j] > v do
        A[j + 1] = A[j]
        j = j−1
      end while
      A[j+1] = v
    end for
  • Worst Case : The input is in descending order.
  • Average Case : For randomly ordered data, we expect each item to move halfway back.
