Given a digraph, list its vertices in such an order that for every edge in the graph, the vertex where the edge starts is listed before the vertex where it ends. If the graph is a DAG, the topological sort has at least one solution.

    There are two different approaches to solve this problem:
  • DFS Method - Do a depth-first search. Collect the nodes in the order they are popped off of the stack. Reverse the final list.
  • Source Removal Method - Arbitrarily choose a node which has no incoming edges. Delete the node and all edges and add the node to list. Repeat the check and delete process until no nodes are left.
