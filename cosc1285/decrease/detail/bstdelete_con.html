Deletion of a node in a binary search tree is a classic data structure problem. It can be solved using one of three corner cases:

  • Case 1 : The target node has no children.
  • Case 2 : The target node has one child.
  • Case 3 : The target node has two children.

NOTE : This operation does not fall nicely into any of the problem solutions spaces we are exploring (divide and conquer, decrease and conquer, etc.)
