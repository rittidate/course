Breadth-First Search (BFS)

Breadth-First Search (BFS) is graph a traversal algorithm which systematically visits each vertex and edge of a graph G.

  • Choose an arbitrary vertex and mark it visited.
  • Add the vertex to a queue.
  • Add each unvisited, adjacent vertex to the queue. If there are several such vertices, break the tie by imposing an ordering on the vertex labels.
  • Continue until the current vertex has no adjacent, unvisited vertices.
  • Remove the vertex from the queue.
  • Starting with the next remaining vertex in the queue, repeat steps 3,4, and 5.
  • The algorithm halts when there are no remaining vertices in the queue.
