Depth-First Search (DFS)

Depth-First Search (DFS) is a graph traversal algorithm which systematically visits each vertex and edge of a graph G.

  • Choose an arbitrary vertex and mark it visited.
  • Proceed to an unvisited, adjacent vertex. If there are several such vertices, break the tie by imposing an ordering on the vertex labels.
  • Continue until a vertex is reached which has no adjacent, unvisited vertices.
  • At each dead-end, backtrace to the last visited vertex and proceed down to the next unvisited, adjacent vertex.
  • The algorithm halts when the starting vertex has no remaining unvisited, adjacent vertices.
  ALGORITHM DFS (G)
/* Implement a Depth First Traversal of a graph. */
/* INPUT : Graph G = ⟨V,E⟩ */
/* OUTPUT : Graph G with its vertices marked with consecutive */ /* integers in initial encounter order. */
  count ← 0
  for i ←0 to v do
    Marked[i] ← 0
  end for
  for i ←0 to v do
  if not Marked[i] then
    DFSR (i)
  end if
  end for

  ALGORITHM DFSR (v)
/* Recursively visit all connected vertices. */
/* INPUT : A starting vertex v */
/* OUTPUT : Graph G with its vertices marked with consecutive */ /* integers in initial encounter order. */
  count ← count + 1
  Marked ← count
  for v′ ∈ V adjacent to v do
    if not Marked[v′] then
      DFSR (v′)
    end if
  end for
  • A DFS search can be implemented with graphs represented as:
  • Adjacency matrices: Θ(V2)
  • Adjacency lists: Θ(V + E)
  • Two distinct orderings of vertices are possible:
  • Pre-order: as vertices are first encountered (pushed onto the stack). For our example: { A, B, C, D, F, E }
  • Post-order: as vertices become dead-ends (popped off the stack).For our example: {F,D,E,C,B,A}
