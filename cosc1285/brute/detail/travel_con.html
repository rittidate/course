Travelling Salesman Problem

Given a collection of n cities and the distance (or cost to travel) between each of them, find the shortest (cheapest) path of visiting all the cities exactly once, and returning to the starting point.

  • The problem statement is deceptively simple, the solution is not.
  • A polynomial solution to this problem would settle the most important open problem in computer science, the P vs NP problem, and could net you 1,000,000 dollars from the Clay mathematics institute!
  • One of the most intensely studied problems in computational mathematics.
  • The problem of finding the shortest Hamiltonian circuit in a weighted connected graph can be reduced to this problem.
  • The first step of the solution for this problem is to calculate the number of tours through n cities.
